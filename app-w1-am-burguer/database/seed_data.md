# seed_data.md
## ¿Para qué sirve este archivo?
Contiene los datos iniciales que se cargan cuando arranca el sistema por primera vez.

## ¿Qué va aquí?
- El menú completo de AM Burguer (hamburguesas, perros, combos, bebidas)
- Precios base
- Categorías de productos
Así no tienes que cargar el menú a mano cada vez que instalas el sistema en un equipo nuevo.
