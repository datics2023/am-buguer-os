# migrations.md
## ¿Para qué sirve este archivo?
Registra todos los cambios que se le hacen a la base de datos con el tiempo.

## ¿Qué va aquí?
- Fecha y descripción de cada cambio al esquema
- Ejemplo: '2025-06-01 — Se agregó campo direccion_guardada a tabla clientes'
Sirve para saber exactamente cómo evolucionó la base de datos y poder revertir cambios si algo falla.
